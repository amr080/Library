Core Payment API Endpoints

Issuance
POST /mint - Mint new USDX {amount, to}
GET /balance - Check USDX balance {address}
GET /totalSupply - Get total USDX supply
GET /sharesOf - Get shares for account {address}

Transfers & Payments
POST /transfer - Basic transfer {to, amount}
POST /transferFrom - Transfer with allowance {from, to, amount}
POST /permit - Gasless approval {owner, spender, value, deadline, signature}
GET /allowance - Check spending allowance {owner, spender}

Yield Features
GET /rewardMultiplier - Current yield rate
POST /addRewardMultiplier - Update yield {increment}
GET /convertToShares - Convert tokens to shares {amount}
GET /convertToTokens - Convert shares to tokens {shares}

DeFi Wrapping
POST /wrap - Convert to wUSDX {amount}
POST /unwrap - Convert back to USDX {amount}

Admin Controls
POST /pause - Emergency pause
POST /unpause - Resume operations
POST /blockAccounts - Block addresses {addresses[]}
