Trading Endpoints

  • POST /orders: Initiate an Order.
  • GET /orders/symbols: Retrieve Active Symbols.
  • GET /orders/symbols/v2: Retrieve Active Symbols v2.
  • GET /orders/{id}: Get Order Details.
  • POST /orders/{id}/cancel: Cancel an Order.
  • GET /orders/status: Check API Status.
  • POST /orders/mutual-funds: Submit a New Order.
  • GET /orders/mutual-funds: Get Mutual Fund Orders.
  • GET /orders/mutual-funds/{id}: Get Mutual Fund Order.
  • GET /orders/mutual-funds/events: List Order Events.
  • POST /orders/mutual-funds/{id}/cancel: Cancel Mutual Fund Order.
  • GET /orders/fixed-income/instruments: Get Tradeable Instruments.
  • GET /orders/fixed-income/instruments/data: Get Instrument Data.
  • GET /orders/fixed-income/instruments/{id}/data: Get Instrument Data by ID.
  • GET /orders/fixed-income/instruments/data/download: Download Instrument Data.
  • GET /orders/fixed-income/instruments/last-update: Get Last Update Timestamp.
  • GET /orders/fixed-income/marks: Retrieve Marks for ISINs/CUSIPs.
  • GET /orders/fixed-income/quote: Get Quote.
  • GET /orders/fixed-income/historic-price: Get Historic Price.
  • POST /orders/fixed-income: Submit Fixed Income Order.
  • GET /orders/fixed-income: Search Fixed Income Orders.
  • GET /orders/fixed-income/{id}: Get Fixed Income Order.
  • DELETE /orders/fixed-income/{id}: Cancel Fixed Income Order.
  • GET /orders/fixed-income/yield-grid: Yield Grid.
  • GET /orders/fixed-income/markup-calculator: Markup Calculator.
  • GET /orders/fixed-income/order-book: Get Full Order Book.
  • PUT /portfolios/settings: Configure Account Settings.
  • GET /portfolios/settings/{account}: Get Account Settings.
  • DELETE /portfolios/settings/{account}: Delete Account Settings.
  • GET /portfolios/settings: Get Settings for All Accounts.
  • POST /portfolios/bundles: Create a Bundle.
  • GET /portfolios/bundles: Get All Bundles.
  • GET /portfolios/bundles/{id}: Get Bundle by ID.
  • PUT /portfolios/bundles/{id}: Update a Bundle.
  • DELETE /portfolios/bundles/{id}: Delete a Bundle.
  • POST /portfolios/models: Create a Model.
  • GET /portfolios/models: Get All Models.
  • GET /portfolios/models/{id}: Get Model by ID.
  • PUT /portfolios/models/{id}: Update a Model.
  • DELETE /portfolios/models/{id}: Delete a Model.
  • GET /portfolios/proposals: Get Proposals.
  • GET /portfolios/proposals/{id}: Get Proposals by ID.
  • GET /portfolios/proposals/errors: Get Analysis Errors.
  • POST /portfolios/proposals/confirm: Confirm Proposals.
  • POST /portfolios/proposals/confirm/multiple: Confirm Multiple Proposals.
  • GET /portfolios/proposals/submissions/{id}: Get Proposals by Submission ID.
  • POST /portfolios/proposals/execute: Submit Proposals for Execution.
  • GET /portfolios/status: Check API Status.
  • POST /portfolios/liquidations: Create Liquidation Request.
  • GET /portfolios/liquidations/{id}: Get Liquidations by ID.
  • POST /portfolios/liquidations/{id}: Update Liquidation Request.
  • POST /portfolios/liquidations/{id}/cancel: Cancel Liquidation Request.
  • POST /portfolios/basket: Create Basket Trade.
  • POST /portfolios/basket/orders: Add Orders to Basket.
  • POST /portfolios/basket/submit: Submit Basket.
  • POST /portfolios/basket/execute: Execute Basket Trade.
  • POST /trading/transactions: Submit Transactions.
  • GET /trading/transactions/status: Get Transaction Status Reports.
  • GET /trading/transactions/statistics: Get Transaction Statistics.
  • GET /trading/transactions/{id}: Get Transaction Status by ID.
  • POST /trading/transactions/{id}/delete: Delete Transactions by ID.
  • GET /trading/status: Check API Status.
