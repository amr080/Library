Authentication & Onboarding
POST /onboard - Register user with KYC
GET /verify - Check compliance status

Core Operations
POST /mint - Create USDX
POST /burn - Redeem USDX
POST /transfer - Send USDX
POST /push - Push to wallet
POST /permit - Gasless approval

Yield Management
GET /yield - View current yield
POST /rebase - Update yield accrual
POST /reinvest - Auto-compound
POST /pay - Schedule payments
POST /withdraw - Extract yield only

Wrapping
POST /wrap - USDX to wUSDX
POST /unwrap - wUSDX to USDX

Monitoring
GET /balance - Check holdings
GET /getYield - View earned yield
GET /history - Transaction history

Admin Controls
POST /pause - Emergency stop
POST /unpause - Resume operations
POST /block - Restrict account
POST /unblock - Remove restriction
POST /setMultiplier - Adjust rates

Status
GET /status - System health
GET /rates - Current yield rates
