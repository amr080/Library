Demo Flow
Onboarding -> KYC
Fiat deposit -> Mint USDX
Push payment to wallet
Show yield accrual

Onboarding & KYC
POST /onboard/register - Register institution
POST /onboard/kyc - Submit KYC
GET /onboard/status - Check KYC status

Fiat Ramps
POST /fiat/deposit - Deposit USD
POST /fiat/withdraw - Withdraw to bank
GET /fiat/status - Check transaction status

USDX Core
POST /mint - Create USDX
POST /burn - Redeem USDX
GET /balance - Check balance
GET /shares - Check shares amount

Transfers & Payments
POST /transfer - Basic transfer
POST /push - Push to wallet
POST /permit - Gasless approval
GET /history - Transaction history

Yield Management
GET /yield/rate - Current yield rate
GET /yield/accrued - Earned yield
POST /yield/reinvest - Auto-compound
GET /yield/forecast - Projected earnings

Cash Management
POST /treasury/deposit - Institutional deposit
POST /treasury/sweep - Auto-sweep yield
GET /treasury/positions - View holdings
