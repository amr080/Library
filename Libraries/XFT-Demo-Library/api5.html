Demo Flow
User onboarding (KYC)
Bank account linking
Fiat deposit -> USDX mint
Show yield accrual
USDX redemption -> Fiat withdrawal

Onramp (Fiat -> USDX)
POST /fiat/deposit - Initiate deposit {amount, currency, paymentMethod}
POST /fiat/deposit - Verify deposit {depositId, kycData}
POST /fiat/deposit - Complete deposit {depositId, paymentConfirmation}
GET /fiat/deposit/status - Check deposit status {depositId}

Offramp (USDX -> Fiat)
POST /fiat/withdraw - Initiate withdrawal {amount, bankDetails}
POST /fiat/withdraw - Verify withdrawal {withdrawId, kycData}
POST /fiat/withdraw - Complete withdrawal {withdrawId}
GET /fiat/withdraw/status - Check withdrawal status {withdrawId}

Support
GET /fiat/supported-methods - List supported payment methods
GET /fiat/limits - View transaction limits
GET /fiat/fees - View transaction fees
GET /fiat/rates - View current exchange rates

KYC/Banking
POST /kyc/verify - Submit KYC verification data
POST /bank/link - Link a bank account
GET /bank/accounts - Retrieve linked bank accounts
